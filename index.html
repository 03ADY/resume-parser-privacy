Privacy Policy: ResumeAI Screener

This Privacy Policy explains how the **ResumeAI Screener** application handles your data when accessing your Google Drive files for resume processing.

Data Collection and Storage

The ResumeAI Screener is a **local-only application** designed to run on a private server or development machine.

  • No User Data is Stored Externally: This application **DOES NOT** store, transmit, or retain any personally identifiable information (PII) or files (resumes) on any external server, cloud service, or database controlled by the developer (AlignMyCareer).
  • Local Processing Only: All processing of resume data, creation of vector embeddings, and generation of candidate profiles occur entirely on your local machine or designated private server.
  • Temporary Data Usage: Your Google Drive authentication tokens are used solely to download the specified files (resumes) and are saved locally (`drive_token.pickle`) only for the purpose of maintaining a session, preventing you from having to sign in repeatedly.
  • No Uploading: The application does not upload any processed data or files back to Google Drive or any other third-party service.

Google User Data Policy Compliance

The ResumeAI Screener accesses the following data scope from your Google Drive:

  • https://www.googleapis.com/auth/drive.readonly: This permission is used **ONLY** to read and download the resume files you explicitly select for processing.
We adhere strictly to Google's Limited Use Policy. We will never transfer, sell, or use the accessed data for serving advertisements.
